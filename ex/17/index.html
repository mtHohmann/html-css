Cores em CSS

Este título está colorido usando cores por nome

Já este está usando cores por código hexadecimal

Este outro utilizando códigos RGB (Red - Green - Blue)

Por fim, este usa códigos HSL (Hue - Saturation - Light)

Também é possível usar transparência, usando o formato RGBA / HSLA ou Hexadecimal com + duas "casas"
