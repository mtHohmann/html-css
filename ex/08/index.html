Negrito / Destaque

Desta forma temos um negrito não semântico: termo em negrito usando a tag <b>, isso significa que essa tag só tem forma e não é semântica como dito, anteriormente.

Nesta frase, o termo a seguir é de forma semântica pois tem significado e é sem forma: termo em negrito, isso é feito utilizando a tag <strong>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Podemos gerar esse texto utilizando a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Podemos gerar esse texto utilizando a tag <ins>

Existe tambem utilizando o sublinhado/inserido a tag <u> mas ela é não semântica, alterando a forma e não o significado do que está contido
